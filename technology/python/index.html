数据导入

xlsx文件

主要使用函数:

pd.read_excel() #路径前面可加 r , 作用是转义,防止\等符号

主要参数:

参数 作用
sheet_name 需要导入数据的工作表表名
header 指定第几行作为表头,若没有表头header=None
names 指定列名称
index_col 指定行索引
usecols 需要读取的列,可以使用列序也可以使用列名
skiprows 需要忽略的行数 (0开始)
nrows 需要读取的行数(从文件头开始)
skip_footer 需要忽略的行数(从最后一行开始)
encoding 编码方式

CSV文件和TXT文件

主要使用函数:

pd.read_csv()

主要参数:

参数 作用
sheet_name 需要导入数据的工作表表名
header 指定第几行作为表头,若没有表头header=None
names 指定列名称
index_col 指定行索引
usecols 需要读取的列,可以使用列序也可以使用列名
skiprows 需要忽略的行数 (0开始)
nrows 需要读取的行数(从文件头开始)
skip_footer 需要忽略的行数(从最后一行开始)
encoding 编码方式 (utf-8, gbk, utf-8-sig)
sep 指定分割符,默认’,',txt文件一般用’\t'或者’ '

mySQL

  • 链接mysql
  • 通过sql语句查询并读入pandas中
import pandas as pd
import pymysql

# 导入sql文件
HOST = 'localhost'
PORT = 3306
USER = 'root'
PASSWORD = ''
DB = ''

# 查询语句
sql = """select * from city"""

# 1. 创建链接
eng = pymysql.connect(host=HOST, user=USER, port=PORT, password=PASSWORD, db=DB)

# 2. 执行sql查询语句
df = pd.read_sql(sql, eng)

DataFrame基础操作

索引,切片

取出某列或多列:

data['列索引名','列索引名'] #或序号
data.A #取出名字叫A的那一列

取出某一行或多行:

data[0:1] #取出第0行 
data[:4]   #取出0,1,2,3行

对于行索引建议使用loc方法和iloc方法:

data.loc['行索引名'] # loc方法使用索引名称
data.iloc[0]; #取出第0行

y对于取出某行某列:

data.iloc[[0,1,2],[2,3,4] #取出0,1,2行的2,3,4列

修改数据

直接通过索引的方式修改:

data.iloc[4,2] =123 #第4行2列修改为123

一个标量修改某一列,则会广播到该列的每一行:

data['索引'] = 15 # 将这一列全变为15

也可以用列表来修改:

data.fl = np.arange(5) #将名为fl的那一列换成 0,1,2,3,4

还可以使用一个Series:

val = pd.Series([1,2,3],index=['a','b','c'])
data.fl = val #将fl列的a,b,c行分别变为1,2,3

修改索引,使用reindex方法:

data2 = data.reindex([1,2,4,5]) #将行索引改为1,2,4,5
s = ['aa','bb','cc']
data.reindex(columns=s) #修改列索引为 'aa','bb','cc'

丢除指轴上的值,但是不改变原本的数据:

data.drop(1) #丢除第1行
data.drop(['A'],axis=1) #丢除索引名为A的列

描述统计

方法名 作用
.head() 展示前几行
.sum() 求和计算
.mean() 平均值计算
.var() 方差计算
.std() 标准差计算

.cov() 协方差矩阵

apply和applymap的使用

f = lambda x: x.max() - x.min()
data.apply(f)   #对每一列进行操作
data.apply(f,axis=1)   #对每一行进行操作

def f(x):
    return pd.Series([x.min(),x.max()],index=['min','max'])
data.apply(f)

format = lambda x:'%.2f'%x 
data.applymap(format) #元素级别的操作用applymap来实现

排序和排名

data.sort_values()

相关参数设置:

参数 作用
axis 选择0,1;0为列,1为行
ascending True为升,False为降,可以列表输入
inplace 是否用排序后的数据替换现有数据
na_position last,first 缺失值排前面还是后面 
data.sort_index(by = 'a') #按a列进行排序
